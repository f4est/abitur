# Edugate - Образовательная платформа для абитуриентов

Edugate (Abiturienty) - это современная веб-платформа, разработанная для помощи абитуриентам в выборе учебного заведения и профессии. Платформа предоставляет удобный каталог образовательных учреждений, инструменты профориентации и актуальную информацию для поступающих.

## Основные возможности

### Для пользователей
1. **Каталог учебных заведений**
   - Просмотр списка с фильтрацией и поиском
   - Детальная информация о каждом учебном заведении
   - Фотогалерея и логотипы
   - Образовательные программы с требованиями к экзаменам
   - Расположение на карте (интеграция с Яндекс.Картами)

2. **Профориентация**
   - Прохождение тестов профориентации
   - Получение рекомендаций по выбору профессии
   - Подбор подходящих учебных заведений

3. **Отзывы и рейтинги**
   - Внутренние отзывы пользователей
   - Интеграция с отзывами из 2GIS
   - Рейтинги учебных заведений

4. **Личный кабинет**
   - Сохранение избранных учебных заведений
   - История прохождения тестов
   - Управление профилем

### Для администраторов
1. **Управление пользователями**
   - Создание, редактирование и удаление пользователей
   - Управление ролями и правами доступа

2. **Управление учебными заведениями**
   - Добавление и редактирование информации
   - Загрузка логотипов и фотографий
   - Структурированная контактная информация
   - Импорт отзывов из внешних источников (2GIS)

3. **Управление образовательными программами**
   - Создание и редактирование программ
   - Настройка требований к экзаменам

4. **Управление контентом**
   - Модерация отзывов пользователей
   - Управление тестами профориентации
   - Аналитика и статистика

## Технический стек

### Frontend
- **Nuxt 3** - фреймворк для создания Vue.js приложений
- **Vue 3** - прогрессивный JavaScript фреймворк
- **Tailwind CSS** - утилитарный CSS фреймворк
- **Яндекс.Карты API** - для интеграции карт

### Backend
- **Node.js** - серверная JavaScript-среда
- **Nuxt API** - API-эндпоинты, встроенные в Nuxt
- **Prisma ORM** - современный ORM для Node.js и TypeScript

### База данных
- **SQLite** - файловая реляционная база данных

### Инфраструктура
- **Git** - система контроля версий
- **npm** - менеджер пакетов

## Установка и настройка

### Предварительные требования
- Node.js (версия 16.x или выше)
- npm (версия 7.x или выше)
- Git

### Шаги по установке

1. **Клонирование репозитория**
   ```bash
   git clone https://github.com/yourusername/abiturienty.git
   cd abiturienty
   ```

2. **Установка зависимостей**
   ```bash
   npm install
   ```

3. **Настройка переменных окружения**
   - Создайте файл `.env` в корне проекта на основе примера из раздела [Руководство по настройке](#руководство-по-настройке)

4. **Инициализация базы данных**
   ```bash
   npx prisma db push
   ```

5. **Заполнение базы данных тестовыми данными (опционально)**
   ```bash
   npx prisma db seed
   ```

6. **Запуск приложения в режиме разработки**
   ```bash
   npm run dev
   ```

7. **Доступ к приложению**
   - Откройте браузер и перейдите по адресу: `http://localhost:3000`
   - Для доступа к админ-панели используйте: `http://localhost:3000/admin`

8. **Доступ к Prisma Studio (управление базой данных)**
   ```bash
   npx prisma studio
   ```
   - Откройте браузер и перейдите по адресу: `http://localhost:5555`

### Сборка для продакшн
```bash
npm run build
npm run start
```

## Руководство по настройке

### Переменные окружения (.env файл)

Создайте файл `.env` в корне проекта со следующим содержимым:

```
# Основные настройки
NODE_ENV=development
HOST=localhost
PORT=3000

# База данных
DATABASE_URL="file:./prisma/dev.db"

# JWT Authentication
JWT_SECRET="ваш_секретный_ключ_для_jwt_токенов"
JWT_EXPIRY="7d"

# API ключи внешних сервисов
YANDEX_MAPS_API_KEY="ваш_api_ключ_яндекс_карт"
TWOGIS_API_KEY="ваш_api_ключ_2gis" # Опционально

# Настройки загрузки файлов
UPLOAD_DIR="public/uploads"
MAX_FILE_SIZE=5000000 # 5MB
```

### Получение API ключей

#### Яндекс.Карты API

1. Перейдите на [developer.tech.yandex.ru](https://developer.tech.yandex.ru/)
2. Зарегистрируйтесь или войдите в личный кабинет
3. Создайте новое приложение в разделе "JavaScript API и HTTP Геокодер"
4. Получите API-ключ и добавьте его в файл `.env`

#### 2GIS API (опционально)

1. Перейдите на [dev.2gis.ru](https://dev.2gis.ru/)
2. Зарегистрируйтесь или войдите в личный кабинет
3. Создайте новое приложение
4. Получите API-ключ и добавьте его в файл `.env`

### Настройка базы данных

## API и внешние сервисы

### Внутренние API эндпоинты

- `/api/auth/*` - Авторизация и аутентификация
- `/api/users/*` - Управление пользователями
- `/api/schools/*` - Управление учебными заведениями
- `/api/programs/*` - Управление образовательными программами
- `/api/test-api/*` - API для тестов профориентации
- `/api/upload/*` - Загрузка файлов
- `/api/external-reviews/*` - Работа с внешними отзывами

### Интеграции с внешними сервисами

1. **Яндекс.Карты**
   - Используется для отображения расположения учебных заведений
   - Требуется API-ключ в .env файле

2. **2GIS (опционально)**
   - Используется для импорта отзывов о учебных заведениях
   - Требуется API-ключ в .env файле

## Разработка

### Команды

- `npm run dev` - Запуск сервера разработки
- `npm run build` - Сборка приложения для продакшн
- `npm run start` - Запуск собранного приложения
- `npm run lint` - Проверка кода линтером
- `npx prisma studio` - Запуск GUI для работы с базой данных
- `npx prisma db push` - Синхронизация схемы базы данных
- `npx prisma db seed` - Заполнение базы данных тестовыми данными

### Добавление новых возможностей

1. Для добавления нового API эндпоинта создайте файл в папке `server/api/`
2. Для добавления новой страницы создайте файл в папке `pages/`
3. Для добавления нового компонента создайте файл в папке `components/`
